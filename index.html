La palabra inglesa "FRAMWORK" (marco de trabajo) define, en terminos generales, un conjunto de conceptos, prácticas y criterios para enfocar un tipo de problematica particular que sirve como referencia, para enfrentar y resolver problemas de indole similar.

En el desarrollo de software, un framework o infrestructura digital, es una estructura conceptual y tecnologica de soporte definido, normalmente con artefactos o moculos de software concretosm, que puede servir de base para la organización y desarrollo de software. Tipicamente hablanco, puede incluir soporte de programas, bibliotecas, y un lenguaje interpretado, entre otras herramientas, para asi ayudar a desarrollar y unir componentes de un proyecto.
